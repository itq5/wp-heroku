一个简单的短代码插件

== Description ==

支持的功能
1.短代码，移植自CorePress 主题
2.支持让gravatar头像替换为国内源
3.支持使用QQ邮箱的用户头像调整为QQ头像
4.支持缓存QQ头像（如果头像源出现禁止跨站访问~）
